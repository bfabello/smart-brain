# Smart Brain 

Smart Brain is full stack web application that uses an image recognition API to detect faces


## Deploy

You must add your own API key in the src/App.js file to connect to Clarifai API

```bash
npm install
npm start

```

# Technologies Used
## Front End
- ReactJS 
- Javascript
- HTML5
- CSS3
- Clarifai API


## Back End
- Node.js
- Express.js
- PostgreSQL
